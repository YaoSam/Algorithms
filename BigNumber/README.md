#万进制的高精度浮点数

##基本实现##
模仿计算机内部的浮点数。但是采用了万进制（**烦死人了**）  
除了数据段之外有4个变量，**length**录数据段的长度，**point**记录小数点的位置，或者说\*10^n，
**sign**记录正负，**datasize**记录数组的长度。

------
##使用说明##
###初始化###
主要通过字符串输入输出。当然直接输入浮点数或者整数也是可以的。
>R one("-1234567890.123456789123123");  
>R two(123.432);  
>R three(-1234);  
>R four; cin>>four;

###运算符###
目前只做了**\+**，**\-**，**\***。
目标：  
- 各种比较运算符
- 除法（这里涉及到一个精确度的问题）
- 取模%
